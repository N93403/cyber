## ⚠️ AVVERTENZE IMPORTANTI
- **USO ESCLUSIVAMENTE DIDATTICO**
- Solo in ambienti controllati e isolati
- Richiede consenso esplicito per il testing

## Descrizione
Laboratorio completo per lo studio di:
- Attacchi Denial of Service (HTTP Flood)
- Architetture botnet e Command & Control
- Tecniche di monitoraggio e mitigazione

## Requisiti
- VirtualBox o VMware
- 3 VM: Kali Linux, Metasploitable, Ubuntu Server
- 4GB+ RAM totale
- Rete NAT isolata
